# Suji Worked Bridge: Prime Factorization Entry Webtoon

## Purpose
This is a webtoon-style variant of the prime factorization entry bridge.

It keeps the same learning goal:
- help Suji find the first starting point for `24`
- do not push into full factorization too early

## Short Scene

### 1. Problem Prompt
- Panel caption: "`24`를 소인수분해하기 전, 먼저 어떤 수로 나누고 싶은지 말해 보자."

### 2. Likely Frozen Response From Suji
- Suji: "어디서부터 해야 할지 모르겠어..."
- Suji's thought bubble: "그냥 아무 수나 나누면 안 될 것 같은데."

### 3. Tutor's First Question
- Tutor: "가장 먼저 나눠 볼 쉬운 수는 뭐니?"

### 4. One Small Hint
- Tutor: "끝자리가 짝수면 `2`부터 볼 수 있어."
- Suji's thought bubble: "`24`는 끝자리가 `4`네. 그러면 `2`를 먼저 볼 수 있겠다."

### 5. Good Stopping Point
- Suji: "`24`는 짝수니까 `2`부터 나눠 보고 싶어."
- Stop the scene here. The goal is that Suji sees the first entry point, not that she finishes the whole factorization yet.
