# mastery-review

Use this skill to review mastery-related logic before changing learner progression behavior.

## Check
- Do not silently change mastery thresholds or pass criteria.
- Verify prerequisite assumptions match the intended learning path.
- Flag recommendation logic that overpromises confidence or skips required validation.
