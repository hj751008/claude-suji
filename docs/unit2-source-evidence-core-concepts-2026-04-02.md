  choices are not positive integers.

### `U2-S2` number-line comparison, midpoint, and ordering

- 미래엔 중단원마무리 page 1 item 03 asks for the number corresponding to the
  midpoint between two marked values on a number line.
- 미래엔 중단원마무리 page 2 includes statement-check items about the largest
  or smallest signed number, how many values exceed a given negative integer,
  and how to read inequality language for signed values.
- 미래엔 중단원마무리 answer/explanation page 4 states that the midpoint item
  should be solved by locating the point halfway between two marked values on
  the number line.
- These sources justify the current `U2-S2` wording around left-right position,
  midpoint, interval, and inequality reading.

### `U2-S3` absolute value as distance

- 미래엔 중단원마무리 page 1 includes items about ordering numbers by distance
  from the origin and about checking absolute-value statements.
- 미래엔 중단원마무리 answer/explanation page 4 explains the distance-order
  item by ordering numbers according to the size of their absolute values.
- The same explanation page also includes an absolute-value note that the
  smallest absolute value belongs to zero and then lists integers whose absolute
  value stays within a bound.
- 발전 문제은행 page 2 also includes a signed-number task where absolute values
  constrain possible integer values, which reinforces that Unit 2 treats
  absolute value as a distance/size condition rather than as a copied sign.
