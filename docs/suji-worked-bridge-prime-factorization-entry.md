# Suji Worked Bridge: Prime Factorization Entry

## Purpose
This short bridge connects one Unit 1 sample problem to one tutor dialogue flow so the learning experience is easier to use in practice.

It is tailored to Suji's main difficulty:
- weak starting point

## Connected Example

### 1. Problem Prompt
- "`24`를 소인수분해하기 전, 먼저 어떤 수로 나누고 싶은지 말해 보자."

### 2. Likely Frozen Response From Suji
- "어디서부터 해야 할지 모르겠어."

### 3. Tutor's First Question
- "가장 먼저 나눠 볼 쉬운 수는 뭐니?"

### 4. One Small Hint
- "끝자리가 짝수면 `2`부터 볼 수 있어."

### 5. Good Stopping Point
- Suji says she wants to start with `2` and explains that `24` is even, so `2` is an easy first check.

## Use Note
- Stop here if Suji finds the first entry point by herself.
- Do not rush into the full factorization if the goal is only to rebuild the starting habit.
