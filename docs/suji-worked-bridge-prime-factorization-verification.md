# Suji Worked Bridge: Prime Factorization Verification

## Purpose
This short bridge connects one Unit 1 sample problem to one tutor dialogue flow for `S4 Verification Habit`.

It is tailored to Suji's main difficulty in this stage:
- unstable procedure
- low confidence when checking whether an answer is really complete

## Connected Example

### 1. Problem Prompt
- "`12`의 소인수분해 답으로 `2 x 6`과 `2 x 2 x 3`이 있다. 어느 쪽이 맞는지 이유와 함께 말해 보자."

### 2. Likely Frozen Response From Suji
- "둘 다 `12`는 되니까 뭐가 다른지 잘 모르겠어."

### 3. Tutor's First Question
- "곱이 맞는 것 말고, 먼저 각 인수가 소수인지 볼까?"

### 4. One Small Hint
- "`6`이 소수인지 먼저 확인해 보면 두 답의 차이가 보여."

### 5. Good Stopping Point
- Suji says `2 x 6`은 `6`이 소수가 아니라서 소인수분해가 끝난 답이 아니고, `2 x 2 x 3`이 맞는 답이라고 말한다.

## Use Note
- Keep the check narrow: first see whether every factor is prime.
- Do not add full back-check or extra application steps yet if Suji can already reject `2 x 6` for the right reason.
