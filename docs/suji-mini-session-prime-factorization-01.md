# Suji Mini Session: Prime Factorization 01

## Purpose
This is the first short tutoring session plan for Suji on Unit 1 `소인수분해`.

It is designed for Suji's current profile:
- weak starting point
- weak mixed-concept handling
- unstable procedure
- low application confidence

Session length:
- about 10-15 minutes

## 1. Clear Session Goal
Help Suji do two things without overload:
- find the first starting point for a simple prime factorization problem
- notice that a final answer must contain only prime factors

This session does not aim to cover full application problems.

## 2. Activities

### Activity 1. Entry Start Practice
- Style:
  - `webtoon`
- Basis:
  - `docs/suji-worked-bridge-prime-factorization-entry-webtoon.md`
- Purpose:
  - reduce freezing and help Suji see the first move on `24`
- Prompt:
  - "`24`를 소인수분해하기 전, 먼저 어떤 수로 나누고 싶은지 말해 보자."
- First tutor question:
  - "가장 먼저 나눠 볼 쉬운 수는 뭐니?"
- Likely stuck point:
  - Suji says she does not know where to begin
- Small support:
  - remind her to look at whether the number is even
- Success for this activity:
  - Suji says she wants to start with `2` because `24` is even

### Activity 2. One Simple Continue Step
- Style:
  - `plain`
- Basis:
  - `S2` problem/example logic from the Unit 1 docs
- Purpose:
  - move only one step past the start without turning the session into a long procedure drill
- Prompt:
  - "`24`를 `2 x 12`로 나누었다면, 아직 더 쪼갤 수 있는 수가 있을까?"
- First tutor question:
  - "지금 남은 수들 중에서 아직 소수가 아닌 것은 뭐니?"
- Likely stuck point:
  - Suji thinks `2 x 12` is already enough because she has started successfully
- Small support:
  - keep the question narrow and ask only about which number is still not prime
- Success for this activity:
  - Suji points to `12` and says it can still be broken further

### Activity 3. Verification Habit
- Style:
  - `plain`
- Basis:
  - `docs/suji-worked-bridge-prime-factorization-verification.md`
- Purpose:
  - build one simple checking habit without requiring full back-check
- Prompt:
  - "`12`의 소인수분해 답으로 `2 x 6`과 `2 x 2 x 3`이 있다. 어느 쪽이 맞는지 이유와 함께 말해 보자."
- First tutor question:
  - "곱이 맞는 것 말고, 먼저 각 인수가 소수인지 볼까?"
- Likely stuck point:
  - Suji says both answers seem fine because both make `12`
- Small support:
  - ask her to inspect only `6` first
- Success for this activity:
  - Suji says `2 x 6`은 `6`이 소수가 아니라서 끝난 답이 아니고, `2 x 2 x 3`이 맞다고 말한다

## 3. Format Choice In This Session
- Use `webtoon` only in Activity 1 because the biggest risk there is freezing before the first move.
- Use `plain` in Activities 2 and 3 because these are procedure and checking steps, where extra scene elements may add noise.

## 4. What Counts As Success For Today
- Suji starts `24` by choosing `2` for a clear reason.
- Suji notices that `12` is still not prime in `2 x 12`.
- Suji rejects `2 x 6` as a final prime factorization because `6` is not prime.

If these three things happen, the session is successful even if Suji does not complete a long full-solution sequence.

## 5. Where To Stop Before Overload
- Stop after Activity 3 if Suji has shown one stable start and one stable verification judgment.
- Do not move into divisor count, square-making, GCD/LCM, or a long mixed application task.
- If Suji freezes hard in Activity 2, end after a successful Activity 1 instead of pushing forward.
