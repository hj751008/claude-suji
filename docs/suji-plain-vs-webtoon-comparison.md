# Suji Plain vs Webtoon Comparison

## 1. Purpose Of The Comparison
This document compares the plain bridge and the webtoon-style bridge for the same Unit 1 entry task.

The goal is not to change the math. The goal is to choose the format that helps Suji see the first starting point more easily.

Basis:
- `docs/suji-worked-bridge-prime-factorization-entry.md`
- `docs/suji-worked-bridge-prime-factorization-entry-webtoon.md`

## 2. When Plain Style Is Better For Suji
- when the target is one clean procedure step
- when Suji already understands the scene and only needs the next action
- when mixed presentation might distract her from the math cue
- when the tutor wants the shortest path from prompt to first step

## 3. When Webtoon Style Is Better For Suji
- when Suji freezes before writing the first line
- when she needs the starting clue to feel more visible and less abstract
- when low confidence makes a direct worksheet tone feel heavy
- when a short thought bubble can show the difference between panic and a usable first move

## 4. Risks Of Each Style
- Plain style risk: it may feel too bare if Suji is anxious and cannot see where to begin.
- Plain style risk: it may help with one step but not reduce emotional freezing.
- Webtoon style risk: if overused, Suji may focus on the scene more than the procedure.
- Webtoon style risk: extra presentation can make mixed-concept problems feel busier.

## 5. Simple Decision Criteria For Choosing One
- Choose plain style if the goal is to stabilize one procedure step that Suji can already partially see.
- Choose webtoon style if the main problem is that Suji cannot start at all.
- Choose plain style for verification or cleanup steps.
- Choose webtoon style for entry moments, hesitation points, or confidence-sensitive starts.

## 6. One Default Rule For Future Content
- Default to plain style first.
- Use webtoon style only for the first entry point of a skill, or when Suji repeatedly freezes before the first step.

This default fits Suji because her biggest weakness is often the start, but too much presentation can add noise once procedure practice begins.
