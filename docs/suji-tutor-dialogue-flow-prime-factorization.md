# Suji Tutor Dialogue Flow: Prime Factorization

## Purpose
This document defines short, practical tutor dialogue flows for Unit 1 `소인수분해`.

It is based on:
- `docs/suji-subskills-prime-factorization.md`
- `docs/suji-error-patterns-prime-factorization.md`
- `docs/suji-learning-principles.md`

It is tailored to Suji's current difficulty profile:
- weak starting point
- weak mixed-concept handling
- unstable procedure
- weak application confidence

This document does not define curriculum codes, numeric mastery thresholds, or scoring cutoffs.

## S1 Concept Check: prime, composite, or neither

- When this flow should be used:
  - Suji confuses `1`, prime numbers, composite numbers, or divisor language before calculation starts.
- Tutor's first question:
  - "이 수의 약수를 먼저 써 보면 몇 개일까?"
- Likely student response from Suji:
  - "`1`도 나누어지니까 소수인 것 같아."
- Tutor's next guiding question:
  - "소수는 약수가 정확히 어떤 두 개여야 하지?"
- Small hint if Suji is still stuck:
  - "소수는 `1`하고 자기 자신, 이렇게 두 약수가 있어야 해."
- Good stopping point:
  - Suji says `1` is neither prime nor composite and gives a divisor-based reason.
- Confidence line:
  - "지금처럼 이름보다 기준을 먼저 보면 덜 헷갈려."

## S1 Vocabulary Repair: factor, prime factor, divisor, multiple

- When this flow should be used:
  - Suji mixes up math words and answers with the wrong concept type.
- Tutor's first question:
  - "지금 네가 말한 것은 수의 종류일까, 관계 이름일까?"
- Likely student response from Suji:
  - "약수랑 소수가 비슷한 말 아닌가?"
- Tutor's next guiding question:
  - "소수는 수의 종류이고, 약수는 어떤 수에 대한 관계라는 점이 어떻게 다를까?"
- Small hint if Suji is still stuck:
  - "소수는 숫자 자체의 성질이고, 약수는 다른 수를 나누는 역할이야."
- Good stopping point:
  - Suji can sort one example into the right word type before solving.
- Confidence line:
  - "단어를 구분하면 다음 계산이 훨씬 가벼워져."

## S2 Entry Flow: starting with a divisor pair

- When this flow should be used:
  - Suji freezes at the first step or splits the number randomly.
- Tutor's first question:
  - "가장 먼저 나눠 볼 쉬운 수는 뭐니?"
- Likely student response from Suji:
  - "어디서부터 해야 할지 모르겠어."
- Tutor's next guiding question:
  - "짝수인지, `3`으로 나뉘는지처럼 쉬운 것부터 확인해 볼까?"
- Small hint if Suji is still stuck:
  - "끝자리가 짝수면 `2`부터 볼 수 있어."
- Good stopping point:
  - Suji chooses one reasonable divisor pair and explains why she started there.
- Confidence line:
  - "첫 줄만 잡히면 그다음은 훨씬 따라가기 쉬워."

## S2 Continuation Flow: keep splitting until all factors are prime

- When this flow should be used:
  - Suji stops too early or leaves a composite number in the result.
- Tutor's first question:
  - "지금 남은 수들 중에서 아직 소수가 아닌 것은 뭐니?"
- Likely student response from Suji:
  - "`2 x 6`이면 된 것 같은데?"
- Tutor's next guiding question:
  - "`6`은 소수일까, 아니면 한 번 더 나눌 수 있을까?"
- Small hint if Suji is still stuck:
  - "소인수분해의 끝은 남은 수가 전부 소수일 때야."
- Good stopping point:
  - Suji continues one more split by herself and reaches all-prime factors.
- Confidence line:
  - "끝까지 보는 습관이 생기면 실수가 많이 줄어."

## S3 Expression Flow: rewrite the final answer cleanly

- When this flow should be used:
  - Suji's work is partly right, but the final written expression is unstable.
- Tutor's first question:
  - "마지막 답 줄에는 무엇만 남아 있어야 하지?"
- Likely student response from Suji:
  - "위에 있던 걸 그냥 그대로 쓰면 되는 거 아니야?"
- Tutor's next guiding question:
  - "위에서 나온 소수만 골라서 한 줄로 다시 쓰면 어떻게 되니?"
- Small hint if Suji is still stuck:
  - "중간 가지는 지우고, 마지막에는 소수만 모두 모아 써 봐."
- Good stopping point:
  - Suji writes one complete prime factorization without a composite factor or missing factor.
- Confidence line:
  - "지금 답 정리가 훨씬 안정적이야."

## S3 Comparison Flow: same answer from different split paths

- When this flow should be used:
  - Suji thinks different split orders mean different final answers.
- Tutor's first question:
  - "두 답에 들어 있는 소수 종류와 개수는 같니?"
- Likely student response from Suji:
  - "나누는 순서가 달라서 다른 답 같아."
- Tutor's next guiding question:
  - "모양 말고, 마지막에 남은 소수들을 하나씩 비교하면 어떨까?"
- Small hint if Suji is still stuck:
  - "순서는 달라도 같은 소수들이 같은 개수로 있으면 같은 답이야."
- Good stopping point:
  - Suji accepts that two different paths can produce the same final factorization.
- Confidence line:
  - "절차가 달라도 핵심 구조를 볼 수 있으면 훨씬 강해져."

## S4 Verification Flow: check if the answer is really complete

- When this flow should be used:
  - Suji accepts a partly correct answer or cannot choose between similar answers.
- Tutor's first question:
  - "이 답을 볼 때 먼저 확인할 것은 무엇일까?"
- Likely student response from Suji:
  - "곱해서 맞으면 맞는 거 아니야?"
- Tutor's next guiding question:
  - "곱이 맞는 것 말고, 소수만 있는지와 빠진 것이 없는지도 봐야 하지 않을까?"
- Small hint if Suji is still stuck:
  - "순서를 이렇게 잡아 봐: 소수만 있니, 빠진 건 없니, 원래 수로 다시 맞니."
- Good stopping point:
  - Suji uses the full check order once and rejects an incomplete answer with a reason.
- Confidence line:
  - "지금은 찍는 게 아니라 확인하면서 풀고 있어."

## S4 Back-Check Flow: product check without overload

- When this flow should be used:
  - Suji knows she should check, but loses confidence when asked to multiply back.
- Tutor's first question:
  - "전부 한 번에 계산하지 말고, 어디부터 확인하면 덜 부담될까?"
- Likely student response from Suji:
  - "검산하려면 또 다 계산해야 해서 헷갈려."
- Tutor's next guiding question:
  - "같은 소수끼리 먼저 묶거나, 한 부분씩 곱해 보면 어떨까?"
- Small hint if Suji is still stuck:
  - "`2 x 2 x 3`이면 `4 x 3`처럼 가까운 묶음부터 볼 수 있어."
- Good stopping point:
  - Suji completes one simple back-check without stopping from anxiety.
- Confidence line:
  - "검산을 작게 나누면 충분히 해낼 수 있어."

## S5 Transfer Flow: divisor-count application

- When this flow should be used:
  - Suji restarts from zero or lists divisors one by one instead of reusing factorization.
- Tutor's first question:
  - "이 문제에서 필요한 건 소인수 자체일까, 약수의 개수일까?"
- Likely student response from Suji:
  - "그냥 약수를 다 써 봐야 할 것 같아."
- Tutor's next guiding question:
  - "이미 한 소인수분해에서 무엇을 가져오면 더 빨리 시작할 수 있을까?"
- Small hint if Suji is still stuck:
  - "새로 다 쓰기 전에, 먼저 소인수분해 결과를 옆에 적어 두자."
- Good stopping point:
  - Suji starts the application problem from a relevant factorization instead of restarting blindly.
- Confidence line:
  - "이전 결과를 다시 쓰기 시작한 게 좋은 변화야."

## S5 Transfer Flow: making a square number

- When this flow should be used:
  - Suji guesses a multiplier randomly or cannot connect factor counts to square structure.
- Tutor's first question:
  - "어떤 소수의 개수가 지금 홀수라서 하나 더 필요할까?"
- Likely student response from Suji:
  - "뭘 곱해야 할지 감이 안 와."
- Tutor's next guiding question:
  - "제곱수가 되려면 소인수 개수는 어떤 모양이어야 하지?"
- Small hint if Suji is still stuck:
  - "제곱수는 각 소수의 개수가 짝수가 되면 돼."
- Good stopping point:
  - Suji names the missing factor from the prime factorization and explains why.
- Confidence line:
  - "필요한 것만 딱 찾아낸 건 좋은 접근이야."

## S5 Transfer Flow: common divisor or common multiple confusion

- When this flow should be used:
  - Suji mixes GCD and LCM ideas once two numbers appear together.
- Tutor's first question:
  - "지금 찾는 건 공통으로 들어가는 쪽일까, 모두를 덮는 쪽일까?"
- Likely student response from Suji:
  - "둘 다 비슷해서 뭐가 다른지 모르겠어."
- Tutor's next guiding question:
  - "두 수에 같이 있는 것을 보는 문제인지, 둘을 모두 포함해야 하는 문제인지 나눠 볼까?"
- Small hint if Suji is still stuck:
  - "공약수 쪽은 함께 겹치는 부분, 공배수 쪽은 둘을 다 담는 부분을 본다고 생각해 봐."
- Good stopping point:
  - Suji can say which structure she is looking for before trying the calculation.
- Confidence line:
  - "문제 종류를 먼저 가르는 힘이 생기고 있어."
