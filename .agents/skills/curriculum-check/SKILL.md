# curriculum-check

Use this skill to review whether a lesson, problem set, or feature is aligned to the intended curriculum source.

## Check
- Do not invent curriculum mappings or achievement standards.
- Confirm unit/topic alignment against explicit source material.
- Flag missing evidence, ambiguous mapping, or mixed-grade/unit scope.
